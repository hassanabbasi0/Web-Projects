Option 2: Detailed Description (For README.md file)

```markdown
# Brew Haven Coffee Shop Landing Page

A modern, responsive landing page for a fictional coffee shop called "Brew Haven." The design focuses on creating a warm, cozy, and welcoming brand experience that appeals to students, professionals, and coffee lovers.

## Project Goal
To showcase a coffee brand with a minimal, elegant, and inviting aesthetic using warm colors like coffee brown, cream, beige, and soft black.

##  Features
- **Sticky Navigation Bar** – Logo and menu (Home, Menu, About, Contact) stay visible while scrolling
- **Hero Section** – Bold headline, description, and two call-to-action buttons with background coffee image
- **Features Section** – Three key advantages with icons and descriptions
- **About Section** – Brand story with a cozy café interior image
- **Menu Preview** – 4 popular drinks with names, descriptions, and prices
- **Call-to-Action Section** – Encourages visitors to order coffee
- **Footer** – Contact information, location, social media links, and copyright

#  Design Highlights
- Coffee-themed color palette (brown, cream, beige, soft black)
- Clean, modern UI with smooth spacing
- Responsive layout for mobile, tablet, and desktop
- Subtle hover effects and animations
- Google Fonts: Poppins
- Font Awesome icons for visual appeal

##  Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (minimal, optional animations)
- Unsplash for high-quality coffee images

##  Responsive Design
The page is fully responsive and adapts to:
- Mobile devices (≤600px)
- Tablets (600px–900px)
- Desktop (≥900px)

##  How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Customize content, colors, or images as needed

## 📄 Sections Included
- Header (Navigation)
- Hero
- Features
- About
- Menu Preview
- Call-to-Action
- Footer

##  Future Improvements
- Add interactive menu filtering
- Include a contact form
- Add animations on scroll

## License
This project is open-source and available for learning and customization.
```
